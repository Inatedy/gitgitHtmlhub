Git Bueno git es una buena forma de trabajar en equipo, nos permite trabajar en equipo y tener un respaldo principal(local) y uno por cada participante, también tener una línea de tiempo, lo cual no nos ofrece cualquier otro editor de texto, tenemos commits para marcar un punto en la línea de texto


los comandos que aprendí fueron

  • INIT: PARA INICIALIZAR UN REPOSITORIO
  • GIT ADD -A : PARA CARGAR TODOS LOS RCHIVOS AL REPOSITORIO
  • GIT GIT COMMIT -M "LO QUE AGREGARAS" : PARA HACER UN COMMIT
  • GIT REMOTE ADD AROGIN LINK.COM : PARA CONECTAR TU GIT CON GITHUB
  • GIT GIT PUSH ORIGIN MASTER : PARA CARGAR TODOS LOS RCHIVOS AL REPOSITORIO EN LINEA
  • GIT LOG : PARA CHECAR TU REPOSITORIO
  • GIT STATUS : PARA VER EL ESTADO DE TU REPOSITORIO, si hay archivos sin subir al repositorio
